Лабораторная работа 2. Django Framework

Цель работы

Изучить основы разработки веб-приложений на Django.
Освоить работу с моделями, представлениями, шаблонами и базой данных PostgreSQL.

Используемые технологии

  • Python 3
  • Django
  • PostgreSQL
  • HTML, CSS
  • Bootstrap
  • GitHub

Описание проекта

В рамках лабораторной работы было разработано веб-приложение для бронирования отелей.
Реализованы:

  • список отелей с пагинацией;
  • страница отеля с номерами;
  • фильтрация номеров;
  • бронирование;
  • личный кабинет пользователя;
  • панель администратора;
  • разделение прав пользователя и администратора.

Структура базы данных

Используются модели:

  • Hotel
  • Room
  • Booking
  • User (стандартная модель Django)

Связи: - Один отель содержит несколько номеров - Один пользователь может иметь несколько бронирований

Основные страницы сайта

  • Главная страница со списком отелей
  • Страница конкретного отеля
  • Страница бронирования
  • Мои бронирования
  • Админ-панель

Результат работы

В результате работы было создано полноценное веб-приложение с подключённой базой данных PostgreSQL, системой авторизации и возможностью бронирования номеров.
